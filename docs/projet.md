# 📖 Présentation du projet

## 🧠 Contexte

Dans de nombreuses organisations, le suivi de projet est mal structuré :
- informations dispersées
- oublis de tâches
- retards

Le projet Nudge vise à centraliser ces informations et automatiser le suivi.

---

## 🎯 Objectifs

### Objectif principal
Créer une application desktop de suivi de projet avec relance automatique.

### Objectifs fonctionnels
- gestion des projets
- gestion des tâches
- gestion des utilisateurs
- suivi des échéances
- relances automatiques
