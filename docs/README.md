# 📌 Nudge – Logiciel de suivi de projet

## SAE 2.03 – BUT Réseaux & Télécommunications

Nudge est une application de bureau permettant de :
- gérer des projets
- suivre des tâches
- automatiser des relances par mail

## 🎯 Objectif
Améliorer le suivi de projet et éviter les oublis grâce à l’automatisation.

## 🛠️ Stack
- Python 3
- PyQt6
- SQLite
- APScheduler
- SMTP
